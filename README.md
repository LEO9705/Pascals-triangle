# Pascal’s Triangle (C++)

## What it does
Prints Pascal’s Triangle with **height = 5** (starting with 1).  
This version is fixed-size because I haven’t learned arrays/vectors yet.

## Requirements
A C++ compiler (g++ or clang++)

## How to run (Windows, MinGW)
```bash
g++ pascal_triangle.cpp -o app.exe
```
#Command Prompt:
```bash
app.exe
```
#PowerShell:
```bash
.\app.exe
```
##macOS/LINUX:
```bash
g++ pascal_triangle.cpp -o app
./app
```
